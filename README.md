# Sistema_Operacional_Anotacoes

## Ementa e Estrutura de Estudos

1. **Base e Arquitetura:** Conceitos básicos e revisão de arquitetura de computadores.
2. **Processos e Concorrência:** Modelo de processos, comunicação (IPC), escalonamento e programação concorrente.
3. **Memória:** Gerenciamento direto, swapping, memória virtual e troca de páginas.
4. **E/S e Arquivos:** Princípios de hardware/software, interfaces, organização, confiabilidade e desempenho.
5. **Virtualização:** Tipos, construção e implementação de máquinas virtuais.
6. **Kernel Linux (Hands-on):** Bootloader, syscalls, espaço de usuário x kernel, alocadores de memória e drivers de dispositivo.
7. **Segurança:** Randomização de Endereços (ASLR), Position Independent Code (PIC), Buffers/Canários, Bit "Não Executar" (NX) e tabela de páginas isoladas.
